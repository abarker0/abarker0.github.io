Antivirus Cybersecurity Research Project

Honeypot Lifecycle Specialist

This antivirus research project was done as part of one of UMD's ACES courses. Our team studied the impact of antiviruses and antivirus warning banners on attacker behavior by creating four SSH-vulnerable honeypots. This was implemented by creating Linux containers hosted on an ACES virtual machine and using a JavaScript Man-In-The-Middle client to connect attackers to the honeypot after a few password attempts.

Features:

  • Captured attacker IPs, commands entered, and preserved sessions for them to return
  • Automated attack detection, attacker entry, honeypot recycling, and redeployment via Bash scripting
  • Integrated Discord webhook to receive live notifications on attacks

CherryBot

Project Lead

CherryBot is a Discord bot created using Python, the asyncio library, and the Discord.py API. It receives user input by sending messages to the bot, which are then processed asynchronously and sends a message back to the user. This allows multiple different servers to interact with one running instance of the bot.

Features:

  • Tracks user messages sent and displays leaderboard for each server.
  • Manages virtual economy, with currency that can be collected daily, transferred between users, or used for games.
  • Supports games such as blackjack, processing user input through message reactions, managing different game sessions simultaneously, and integrating with the virtual economy to win or lose currency.
  • Plays music when invited to a voice channel by searching for and streaming audio from YouTube videos.
